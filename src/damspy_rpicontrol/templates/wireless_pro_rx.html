damspy-rpicontrol

Manual LAN control page for: RODE Wireless PRO RX 0058 (wireless pro rx).

This older product keeps PA always on. RF start uses a single HID command with antenna ID embedded in byte 4 and signed power encoded into byte 5.

Devices: __DEVICE_NAV__

RF Start

wirepro_freq is the final two digits of the RF frequency from the Wireless Pro spec. Example: 2478 MHz -> 78.

wirepro_power is the exact Wireless Pro dBm value from the spec.

Valid power settings: 4, 3, 0, -4, -8, -12, -16, -20, -40. Negative values are encoded as two's complement in the final HID byte.

Raw RF Start

This sends only the RF start bytes: 15 3 0 freq ant_id pwr.

Command Shape

Documented continuous-wave command format:
